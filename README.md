# CV Perception Systems

## Dev/Creator = tubakhxn

This repository contains three advanced Computer Vision projects focused on perception, tracking, analytics, and intelligent scene understanding using Python and modern AI frameworks.

All projects are built for learning, experimentation, visualization, and portfolio development.

---

# 1. Urban Digital Twin Perception Engine

### Overview

An AI-powered smart-city perception system that transforms traffic, CCTV, drone, or dashcam footage into real-time urban intelligence.

### Features

* Vehicle detection
* Pedestrian detection
* Multi-object tracking
* Traffic density analysis
* Crowd monitoring
* Activity heatmaps
* Congestion estimation
* Trajectory visualization
* Urban analytics dashboard

### Run

```bash
python urban_digital_twin.py
```

---

# 2. Formula 1 Race Intelligence System

### Overview

A motorsport analytics platform that analyzes race footage and generates real-time racing intelligence.

### Features

* Race car detection
* Multi-object tracking
* Racing line visualization
* Overtake detection
* Position tracking
* Speed estimation
* Race heatmaps
* Track activity analysis
* Live race dashboard

### Run

```bash
python formula1_race_intelligence.py
```

---

# 3. Autonomous Drone Airspace Intelligence System

### Overview

A computer vision platform for detecting, tracking, and analyzing drone activity from a single camera feed.

### Features

* Drone detection
* Drone tracking
* Flight-path visualization
* Trajectory prediction
* Airspace monitoring
* Risk-zone analysis
* Activity heatmaps
* Airspace analytics dashboard

### Run

```bash
python drone_airspace_intelligence.py
```
